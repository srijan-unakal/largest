# Largest Number Program in Java
This Java program determines which number is larger between two numbers entered by the user. The numbers are provided through the console, and the program compares them using conditional statements.

The program runs inside a loop so the user can perform the comparison multiple times. After each execution, the user is asked whether they want to continue or stop the program.

## Functionality
* Accepts two numbers from the user through the keyboard.
* Compares the numbers to determine which one is greater.
* Displays the result on the console.
* Allows the user to repeat the process until they choose to exit.

## Concepts Used
This program demonstrates the following Java concepts:
* User input using the `Scanner` class
* Conditional statements (`if`, `else if`, `else`)
* Looping using `while`
* Character handling for user choices

## Program Flow
1. The program asks the user to enter the first number.
2. The user then enters the second number.
3. The program compares the two numbers.
4. The greater number is displayed on the screen.
5. The user is asked if they want to continue.
6. If the user enters **N**, the program terminates.
